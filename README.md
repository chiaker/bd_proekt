# Система управления финансами

Веб-приложение для управления личными финансами с использованием FastAPI, PostgreSQL и чистого JavaScript.

## Функциональность

- **Просмотр всех сущностей БД**: пользователи, счета, категории, транзакции, бюджеты
- **Редактирование транзакций**: добавление, редактирование, удаление транзакций
- **Отчетные формы**: отчеты по транзакциям и категориям
- **Современный интерфейс**: адаптивный дизайн с красивым UI

## Технологический стек

- **Backend**: FastAPI, SQLAlchemy ORM
- **База данных**: PostgreSQL
- **Frontend**: HTML5, CSS3, чистый JavaScript
- **Стилизация**: современный CSS с градиентами и анимациями

## Установка и запуск

### 1. Установка зависимостей

```bash
pip install -r requirements.txt
```

### 2. Настройка базы данных

1. Установите PostgreSQL
2. Создайте базу данных:
```sql
CREATE DATABASE finance_db;
```

3. Выполните SQL скрипт из файла `bd.txt` для создания таблиц

### 3. Настройка переменных окружения

Создайте файл `.env` в корне проекта:
```
DATABASE_URL=postgresql://username:password@localhost/finance_db
```

### 4. Запуск приложения

```bash
python main.py
```

Приложение будет доступно по адресу: http://localhost:8000

## API Endpoints

### Пользователи
- `GET /users` - получить всех пользователей
- `POST /users` - создать пользователя

### Счета
- `GET /accounts` - получить все счета
- `POST /accounts` - создать счет

### Категории
- `GET /categories` - получить все категории
- `POST /categories` - создать категорию

### Транзакции
- `GET /transactions` - получить все транзакции
- `POST /transactions` - создать транзакцию
- `PUT /transactions/{id}` - обновить транзакцию
- `DELETE /transactions/{id}` - удалить транзакцию

### Бюджеты
- `GET /budgets` - получить все бюджеты
- `POST /budgets` - создать бюджет

### Отчеты
- `GET /reports/transactions` - отчет по транзакциям
- `GET /reports/categories` - отчет по категориям

## Особенности интерфейса

- **Адаптивный дизайн**: работает на всех устройствах
- **Современный UI**: градиенты, тени, анимации
- **Интуитивная навигация**: простое переключение между разделами
- **Формы с валидацией**: проверка данных на клиенте и сервере
- **Уведомления**: успешные операции и ошибки

## Использование

1. Откройте http://localhost:8000 в браузере
2. Используйте навигацию для переключения между разделами
3. Добавляйте данные через формы
4. Просматривайте отчеты в разделе "Отчеты"
5. Редактируйте транзакции с помощью кнопок "Редактировать" и "Удалить"
